# AGENTS.md — TABI Project

Instructions for AI agents (Codex, Claude Code, etc.) working on this repository.

---

## Project Overview

**TABI** is an English-language curation site introducing Japan to the world.  
Target audience: international tourists, Japan enthusiasts, potential visitors.

- Stack: Static HTML generated by PowerShell scripts
- Hosting: Netlify
- Language: **English** (all content and UI)

---

## Role Assignment

| Agent | Responsibility |
|-------|---------------|
| **Codex** | `articles.json` — add and edit article data only |
| **Claude Code** | `styles.css`, `scripts/`, `script.js`, feature development |

---

## Codex Scope: articles.json

Codex is responsible **only** for adding and editing entries in `articles.json`.

### Article Schema

```json
{
  "id": "unique-slug-in-kebab-case",
  "title": "Article Title in English",
  "summary": "2-3 sentence summary. Written in natural English.",
  "category": "travel-guide",
  "tags": ["kyoto", "shrine", "first-time"],
  "publishedAt": "2026-05-20",
  "readingTime": 6,
  "affiliate": false
}
```

### Category Values (use exactly as written)
- `travel-guide`
- `culture`
- `food`
- `things-to-buy`
- `hidden-gems`

### Rules for Codex
1. **Only edit `articles.json`** — never touch HTML, CSS, JS, or scripts
2. All text must be in **natural English** — no machine-translated phrasing
3. `id` must be unique, kebab-case, URL-safe
4. `publishedAt` format: `YYYY-MM-DD`
5. `readingTime` in minutes (integer)
6. Set `"affiliate": true` only for `things-to-buy` category articles
7. Do not add fields not listed in the schema above

## Branch Strategy

```
main              Always deployable — do not commit directly
claude/<task>     Claude Code working branches
codex/<task>      Codex working branches
```

Codex must always work on a `codex/<task>` branch and open a PR to `main`.

---

## Content Guidelines

- Tone: Warm, knowledgeable, insider — not a tourist pamphlet
- Titles: Clear, specific, benefit-driven (avoid clickbait)
- Summaries: Factual, useful, 2-3 sentences max
- Tags: 2-5 per article, lowercase, hyphen-separated

---

## What NOT to Do

- Do not edit `site.config.json`, `styles.css`, `script.js`, or any `.ps1` scripts
- Do not create or delete HTML files
- Do not modify `AGENTS.md` or `CLAUDE.md`
- Do not add Japanese text to any field (this is an English-language site)

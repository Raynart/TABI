# TABI — Discover Japan

> Your guide to the real Japan — travel, culture, and the things worth bringing home.

English-language curation site targeting international visitors and Japan enthusiasts worldwide.

## Stack

- Static HTML (generated by PowerShell scripts)
- Hosted on Netlify
- No frameworks, no build tools

## Structure

```
tabi/
├── articles/          Article pages (generated)
├── categories/        Category pages (generated)
├── tags/              Tag pages (generated)
├── assets/            Images, icons
├── scripts/           PowerShell generation scripts
├── docs/              Workflow docs
├── site.config.json   Site-wide config
├── articles.json      All article data
├── styles.css         Design system
├── script.js          Frontend JS
└── index.html         Top page (generated)
```

## Generate Site

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\generate-site.ps1"
```

## Branch Strategy

```
main              Always deployable
claude/<task>     Claude Code branches
codex/<task>      Codex branches
```

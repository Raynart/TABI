# TABI — Discover Japan

> Your guide to the real Japan — travel, culture, and the things worth bringing home.

English-language curation site for international visitors and Japan enthusiasts worldwide.

**Live site**: `tabi.guide` (pending)  
**Hosting**: Netlify  
**Repo**: `github.com/Raynart/TABI`

---

## Stack

- Static HTML (generated by PowerShell scripts)
- No frameworks, no build tools, no Node.js required
- CSS custom properties design system
- Netlify for hosting + CDN

---

## Project Structure

```
tabi/
├── articles/             Article pages (generated — do not hand-edit)
├── categories/           Category pages (generated)
├── tags/                 Tag pages (generated)
├── assets/
│   └── images/           Site images
├── scripts/
│   ├── generate-pages.ps1    Article/category/tag HTML templates
│   ├── generate-feeds.ps1    sitemap.xml, rss.xml, robots.txt
│   └── generate-site.ps1     Entry point — runs all scripts
├── docs/
│   └── tabi-mockup.html      Design reference mockup
├── CLAUDE.md             Claude Code context (project instructions)
├── AGENTS.md             AI agent instructions + article schema
├── site.config.json      Site-wide config (categories, tags, GA4)
├── articles.json         All article data
├── styles.css            Design system
├── script.js             Top page dynamic rendering
├── index.html            Top page (generated)
├── 404.html              Not found page (generated)
├── netlify.toml          Netlify build + headers config
└── _redirects            Netlify redirect rules
```

---

## Getting Started

### Generate the site

```powershell
powershell -ExecutionPolicy Bypass -File ".\scripts\generate-site.ps1"
```

Run this after any change to `articles.json` or `scripts/`.  
Never hand-edit generated HTML files.
